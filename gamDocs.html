HTML File Structure
For Nantucket E-Books

Three things define a Nantucket e-book: the structure of the HTML file, a stylesheet that manages the appearance of the HTML file, and a script which manages the interactive content of the e-book.

Nantucket e-books are written in a syntax called Shanty. A parsing script called Gam formats the plain text into an HTML file with the appropriate document structure. This is then paired with the stylesheet, and the Ahab script for the correct appearance and interactivity. 

    <!DOCTYPE html> - DEFINING AN HTML DOCUMENT

    <html lang="en"> - LANGUAGE IS DEFINED USING !!LANG. IF A LANGUAGE CODE IS NOT SPECIFIED, IT WILL DEFAULT TO 'EN' FOR ENGLISH
        <head>
            <meta charset="UTF-8"> - ESTABLISHING THE CHARACTER SET
            <title>TITLE</title> - CUSTOM TITLE TAG IS GENERATED USING !!TITLE AND !!AUTHOR TAGS. SEE RULES FOR TITLE TAG
            <meta name="author" content="AUTHOR NAME"> - AUTHOR IS DEFINED USING !!AUTHOR. SEE AUTHOR RULES
        </head>

        <body>
        </body>
    </html>
